# DorkSearch
DorkSearch is a framework meticulously crafted to automate the process of discovering concealed endpoints within a given domain. Leveraging a tailored list of specialized search queries known as 'dorks,' this tool enables effortless exploration of the digital landscape of a domain during the reconnaissance process.
![dorkresearch-result](https://github.com/KenjiDoom/DorkSearch/assets/77996865/3c77ccf5-8465-4029-8412-9b0e4a8019e0)

# Installation Process
#### 1st clone the repo
```
git clone https://github.com/KenjiDoom/DorkSearch.git
```
#### 2nd Install the packages
```
pip install -r requirments.txt
```
#### 3rd Run the program
```
python main.py --url 'target' --output testing --proxy proxy_list.txt --dork dork.txt
````
Now you wait for an output
